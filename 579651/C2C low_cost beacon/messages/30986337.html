Sourcecode-Verwaltung

Posted by Christian Stimming on 31 Mar, 2010
Category: Code

Der Sourcecode für den Server und (später auch) Client ist in einem privaten Repository auf http://github.com . Man besorge sich bitte einen account auf github und sage mit den Account-Namen, woraufhin ich jeden dort mit als “Repository Collaborator” mit eintrage.

Das Sourcecode-Verwaltungssystem ist “git”. Auf Linux installiert man das mit “sudo apt-get install git”; auf Windows installiere man TortoiseGIT http://code.google.com/p/tortoisegit/

Die Server-Implementierung läuft nun mit ein bisschen Beispieldaten auf http://carcomm.cstimming.de – da kann man die Beispieldaten als Tabelle oder als Linien auf der OSM-Karte ansehen.

There are no comments.
